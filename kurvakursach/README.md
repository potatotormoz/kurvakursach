\# Watermark Embedder



Консольное Windows-приложение для добавления данных в конец файлов.



\## Особенности реализации

\- Добавляет маркер "WATERMARK\_START" и данные в конец файла

\- Сохраняет исходное содержимое файла

\- Водяной знак теряется при любом редактировании файла



\## Использование

1. Запустите программу
2. Выберите основной файл через диалоговое окно
3. Выберите файл водяного знака через диалоговое окно
4. Программа создаст файл watermarked\_<имя\_основного\_файла>



\## Формат выходного файла



\[Исходные данные файла]\[WATERMARK\_START]\[Данные водяного знака]



\## Ограничения

\- Выходной файл может не соответствовать формальным спецификациям

\- Для извлечения водяного знака нужна отдельная программа

\- Только для учебных целей
