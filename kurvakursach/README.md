\# Watermark Embedder



Консольное Windows-приложение для встраивания цифровых водяных знаков в файлы изображений.



\## Описание



Программа добавляет произвольные данные (водяной знак) в конец файлов изображений, используя упрощённый контейнерный подход. Создана в рамках курсовой работы по теме "Разработка программы для встраивания цифрового водяного знака в цифровое изображение".



\## Особенности реализации



\- Использует метод дописывания данных в конец файла

\- Добавляет маркер "WATERMARK\_START" для идентификации начала данных водяного знака

\- Сохраняет исходное содержимое файла без изменений

\- Работает с любыми типами файлов как с бинарными контейнерами

\- Имеет графический интерфейс выбора файлов через стандартные диалоговые окна Windows
