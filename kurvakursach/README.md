Watermark Embedder



Программа для встраивания цифрового водяного знака в файлы изображений.



ОПИСАНИЕ

Консольное приложение для Windows, которое добавляет произвольные данные в конец файлов изображений. Использует упрощённый контейнерный подход - дописывает данные после маркера "WATERMARK\_START".



ОСОБЕННОСТИ

\- Сохраняет исходное содержимое файла без изменений

\- Добавляет маркер "WATERMARK\_START" и данные водяного знака в конец файла

\- Работает с любыми форматами файлов

\- Имеет простой консольный интерфейс с диалоговыми окнами выбора файлов



ИСПОЛЬЗОВАНИЕ

1\. Запустите программу

2\. В диалоговом окне выберите основной файл (изображение)

3\. В диалоговом окне выберите файл водяного знака (любой файл)

4\. Программа создаст файл с префиксом "watermarked\_"



ПРИМЕР РАБОТЫ

Выберите основной файл (любой):

Основной файл: C:\\images\\photo.jpg



Выберите файл водяного знака (любой):

Водяной знак: C:\\data\\watermark.txt



Информация о файлах:

&nbsp;  Основной файл: 1048576 байт

&nbsp;  Водяной знак: 1024 байт

ОГРАНИЧЕНИЯ

\- Выходной файл может не соответствовать формальным спецификациям форматов

\- Водяной знак теряется при любом преобразовании файла

\- Требуется отдельная программа для извлечения водяного знака

\- Только для учебных целей
